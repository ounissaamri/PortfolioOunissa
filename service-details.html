ATMOSPHERE SERVICE / REPAIR AUTOs

Période: Juin 2024 - Aujourd’hui

REPAIR AUTO

Depuis juin 2024, je travaille sur une application innovante nommée REPAIR AUTO, destinée à faciliter la réservation de mécaniciens automobiles. Cette application permet aux utilisateurs de réserver un mécanicien pour réparer leur voiture, en choisissant le professionnel le plus proche de chez eux et en sélectionnant le prix qui leur convient. L'application offre également la possibilité de payer en plusieurs fois.

Dans la phase d'avant-projet, j'ai participé aux réunions préparatoires, planifié le projet et mis en place les environnements nécessaires. Pendant la phase de conception, j'ai créé des diagrammes UML, mis en place l'architecture logicielle et modélisé la base de données.

Pour le back-end, j'ai intégré des tokens JWT pour authentifier les utilisateurs et sécuriser l'accès aux routes de l'API. J'ai créé des API REST sécurisées pour offrir un accès contrôlé aux données et stocké les informations dans une base de données NoSQL, MongoDB. J'ai également implémenté des fonctionnalités de paiement avec Stripe, intégré des fonctionnalités d'envoi d'email avec Mailchimp, et géré les rôles et les droits des utilisateurs. J'ai utilisé l'ORM Mongoose pour gérer les opérations CRUD sur MongoDB et développé des API avec le framework Express pour une création rapide et efficace, tout en utilisant Postman pour tester et déboguer les API de manière interactive.
Pour le front-end, j'ai intégré des maquettes avec HTML5 et SCSS, résolu diverses anomalies, et utilisé RXJS ainsi que la librairie Material Angular pour le design graphique. J'ai également implémenté le système d’authentification avec Angular JWT.

Les outils que j'ai utilisés au cours de ce projet incluent Jira pour la gestion de projet, Github pour le contrôle de version, Figma pour la conception des interfaces utilisateur, et Postman pour le test des API.
